# 📚 Sistem Informasi Perpustakaan

Aplikasi manajemen perpustakaan berbasis **Laravel** yang dirancang untuk mengelola data buku, transaksi peminjaman, serta pendataan anggota secara efisien dan terstruktur.

---

## 🚀 Fitur Utama
* **Dashboard Interaktif**: Ringkasan data real-time untuk pemantauan perpustakaan.
* **Manajemen Data Master**: Pengelolaan data Buku, Kategori, Penerbit, dan Anggota.
* **Sistem Transaksi**: Alur Peminjaman dan Pengembalian buku yang mudah digunakan.
* **Sistem Laporan**: Monitoring aktivitas transaksi perpustakaan.
* **Manajemen User**: Pengaturan hak akses admin dan pengguna.
